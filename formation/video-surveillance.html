Vidéo-surveillance

Introduction

J'ai réalisé un stage de 6 semaines chez HEBUS-IP à Evreux. J'étais en charge de mettre en œuvre, à partir de Raspberry Pi, un système de vidéo-surveillance avec consommation du flux à distance. Une partie de mon travail était alors consacrée sur la configuration de ce nano-ordinateur. La deuxième partie consistait à la réalisation de scripts shell afin de rendre le système entièrement autonome et à la création d'un site web hébergé par l'une des Raspberry afin de pouvoir consulter les flux vidéos. La dernière partie de mon travail était la création d'un manuel d'utilisation.

HEBUS-IP

HEBUS-IP est une SARL créée en 2004 et spécialisé dans le service informatique réseaux et télécom. La société prend en charge l'intégralité de la mise en place du réseau des entreprises. Elle fournit aussi des services sur la téléphonie des entreprises, en plus des prestations informatiques et d'infogérances. Pour finir, elle propose de développer les différentes applications de leurs clients.

Matériel

  • Ordinateur Windows
  • Raspberry Pi.
  • Pi camera.
  • Carte SD 8 Go minimum avec Raspbian comme système d'exploitation.
  • Câble Ethernet.
  • Câble micro-USB vers USB pour l'alimentation.
  • Clavier USB.
  • Écran avec entrée DVI.
  • Câble HDMI-DVI.

Configuration

  • Installation de Notepad++ sur Windows pour le développement
  • Installation et configuration de Putty sur Windows pour la connexion à distance en ssh.
  • Installation et configuration de WinSCP sur Windows pour le transfert de fichier en sftp.
  • Installation d'Apache et PHP5 sur Rasbian pour le site web.
  • Installation de VLC sur Rasbian pour le coupler à la commande raspivid afin de canaliser la sortie du flux vidéo.
  • Installation de MPlayer sur Rasbian pour enregistrer un flux vidéo.
  • Configuration de Cron sur Rasbian (table de planification) pour exécuter l'enregistrement d'un nouveau fichier vidéo toutes les heures.
  • Installation et configuration de Samba sur Rasbian pour permettre l'accès au dossier partagé du Windows contenant les vidéos enregistrées.

Script shell

  • Script pour lancer le flux vidéo et le canaliser.
  • Script pour enregistrer le flux vidéo dans le dossier partagé tout en créant les sous-répertoires journaliers nécessaires.
  • Script pour lire un fichier vidéo enregistré selon la date donnée.
  • Script qui supprime les fichiers vidéos les plus anciens si jamais l'espace disque ne permet plus l'enregistrement d'une vidéo d'une heure.
  • Script permettant l'installation des différents composants rendant directement la Raspberry fonctionnelle pour le système de vidéo-surveillance.

Site web PHP

  • Lire plusieurs flux vidéo en temps réel.
  • Choisir le nombre d'objets VLC que l'on veut afficher sur une ligne pour le flux en temps réel.
  • Choisir parmi les ip enregistrée lesquelles seront utilisées pour les flux vidéos en temps réels.
  • Choisir l'ip pour laquelle on veux lire une vidéo déjà enregistrée au lieu de lire le flux.
  • Calendrier et horloge pour remplir plus rapidement les champs destinés à connaître la vidéo voulue.
  • Possibilité d'ajouter une nouvelle ip.
  • Possibilité de supprimer une ip déjà enregistrée.
