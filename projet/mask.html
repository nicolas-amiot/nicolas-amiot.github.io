Mask

Cette fonction permet d'ajouter un masque à un champ de saisie.

Le masque peut être configuré depuis 3 attributs data :
  • Le data-mask précise le masque dont les étoiles sont les caractères saisissables et les autres les caractères fixes.
  • Le data-format permet d'autoriser ou non certains caractères. Il y a 3 formats possibles : numeric, alphabetic et alphanumeric (par défaut alphanumeric).
  • Le data-placeholder permet de changer le placeholder par défaut, l'underscore, pour les caractères saisissables.
    • mask.js
    mask.css
    index.html 
    <input type="text" data-mask="**/**/20**" data-format="numeric" />

    <script>
        $("input[data-mask]").each(function(){
            $(this).mask();
        });
    </script>
