TreeView

Cette fonction permet de construire un TreeView à partir d'une liste.

Le treeview peut être configuré à partir de 3 paramètres :
  • La valeur name permet de spécifier le nom des input du formulaire (par défaut treeview)
  • Si présicer le type permet de transformer les élements du plus bas niveau en checkbox, radio ou link. Sinon ce sera du simple texte.
  • Si la valeur childNumber est à true, le nombre d'élément enfant sera affiché à l'élément parent.
    • treeview.js
    treeview.css
    index.html 
    <ul class="treeview">
        <li>Racine<ul>
            <li>Fruits<ul>
                <li>Pomme</li>
                <li>Banane</li>
                <li>Orange</li>
            </ul></li>
            <li>Légumes<ul>
                <li>Carotte</li>
                <li>Concombre</li>
                <li>Poireau</li>
            </ul></li>
            <li>Épices<ul>
                <li>Curcuma</li>
                <li>Gingembre</li>
                <li>Poivre</li>
            </ul></li>
        </ul></li>
    </ul>

    <script>
        $(".treeview").each(function(){
            $(this).treeview({
                "name": "myTreeView",
                "type": "checkbox",
                "childNumber": true
            });
        });
    </script>
