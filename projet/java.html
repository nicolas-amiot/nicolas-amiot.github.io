Classes utilitaires

Vous trouverez en cliquant sur le titre différentes classes utilitaires en JAVA.

Java EE validateur

Le but de ce projet était de valider les champs lors de la perte de focus et lors de la soumission du formulaire au même endroit et sans rechargement de la page.
Celà permet ainsi de ne pas dupliquer le code dans plusieurs langages mais solicite plus souvent le serveur.

Validation par annotation

Ce projet reprend le principe de l'API Hibernate Validator implémentant Javax Validation pour en faire une version maison.

Téléchargement d'un CSV

Permet de faciliter la création d'un fichier csv ou d'un zip contenant plusieurs csv.
Il existe une implémentation pour sauvegarder le fichier en local et un autre pour les servlets pour mettre à l'utilisateur de le télécharger.
Le fichier est enregistré au fur et à mesure pour éviter le out of memory dans le cas de fichier volumineux.
Pour la version servlet l'utilisateur se vera proposer le fichier dès l'écriture des premières lignes mais devra attendre la fin du téléchargement pour l'ouvrir. Il pourra être averti lorsque celui-ci est terminé.
